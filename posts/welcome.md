# 欢迎来到我的博客

这是我的第一篇博客文章，介绍一下博客的功能和特点。

## 博客特点

- 纯前端实现，无需后端
- 支持 Markdown 格式
- 自动加载文章列表
- 按时间倒序排列
- 支持图片显示

## 技术栈

- HTML5
- CSS3
- JavaScript
- marked.js（Markdown 解析）

## 示例图片

> 图片功能已就绪，将图片放入 `/images/` 文件夹后，在 md 中使用 `![图片](/images/xxx.jpg)` 引用即可。

  console.log("Hello, World!");
}
